# login
基于servlet+jsp+javabean实现的登录页面
## 快速运行
配置 src/com/bjpn/javaweb/resource/prop.properties 修改你的数据库信息，确认能建立连接。

运行sql脚本建立表 t_user.sql

搭建服务器环境,如 idea、tomcat project.

访问项目:localhost:xxx/login/

## 数据库
t_user - 账号密码表
